## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Since this is a private repository, please report any security vulnerabilities directly to our team through the designated internal communication channels.

When submitting a report, please include:
- A clear description of the issue  
- Steps to reproduce the vulnerability  
- Affected versions or package  
- Any known mitigations or recommended fixes (if available)

Our team aims to respond to all vulnerability reports within 7 business days.

### Preferred Languages
We prefer all communications to be in English.
